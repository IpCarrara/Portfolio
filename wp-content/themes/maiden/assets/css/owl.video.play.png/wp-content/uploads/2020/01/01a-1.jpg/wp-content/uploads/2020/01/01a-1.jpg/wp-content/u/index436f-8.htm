CodeLive

CodeLive é um editor de Html, CSS e JavaScript online. Com ele o usuário pode escrever o código e ver o resultado na mesma tela. O site possibilita a configuração de quais painéis o usuário gostaria de manter visíveis para o melhor aproveitamento da tela. Basta apertar o botão no topo da tela correspondente ao campo que o usuário gostaria de retirar ou incluir. O estilo do site foi baseado no inicio dos anos 90 e foi pensado como uma ferramenta simples para estudantes de computação. Desenvolvido para resolução de 1920×1080.

Para visitar o site basta clicar na imagem.

Mais informações sobre o código desse projeto podem ser encontradas no GitHub.
